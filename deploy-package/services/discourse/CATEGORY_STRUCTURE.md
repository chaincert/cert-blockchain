## ğŸ¨ Color Coding System

Use CERT brand colors for visual consistency:

- **Mint (#00FFA3)**: Primary/Important (Announcements, Governance, Trading)
- **Electric (#4D9FFF)**: Community/General (Discussion, Development, Projects)
- **Cyber (#9D00FF)**: Technical/Educational (Support, Tutorials, International)
- **Red (#FF4444)**: Security/Critical (Security category only)

## ğŸ“ Category Settings Template

For each category, configure:

1. **Name**: Clear, descriptive name
2. **Color**: Use CERT brand colors
3. **Description**: Brief explanation of category purpose
4. **Permissions**: Who can create/reply/see topics
5. **Default View**: Latest/Top/Categories
6. **Notifications**: Default notification level
7. **Tags**: Enable/disable tags
8. **Slow Mode**: For high-traffic categories
