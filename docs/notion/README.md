## Importing to Notion

### Option 1: Import Markdown Files

1. Open Notion
2. Click "Import" in the sidebar
3. Select "Markdown & CSV"
4. Choose the `.md` files from this folder
5. Notion will create pages automatically

### Option 2: Copy-Paste

1. Open each `.md` file
2. Copy the entire content
3. Paste into a new Notion page
4. Notion will convert markdown formatting
